## **Název projektu**

**Chovatelská stanice Shiba-Inu Sakura**

## **Žák**
Petr Kozák

## **Stručný popis tématu (3–5 vět)**

Webová prezentace chovatelské stanice zaměřené na plemeno Shiba-Inu. Stránka bude mít podobu one-page webu s několika sekcemi, které představí chov, informace o plemeni, aktuální nabídku štěňat a podmínky jejich prodeje. Obsah bude z velké části načítán dynamicky z JSON souborů pomocí AJAXu. Cílem projektu je vytvořit moderní, přehledný a responzivní web se zaměřením na práci s daty a jednoduchou serverovou komunikaci.

## **Cílová skupina**

Zájemci o koupi štěněte Shiba-Inu, návštěvníci hledající informace o plemeni a lidé, kteří hledají důvěryhodného chovatele.

## **Návrh obsahových sekcí**

1. **O nás** – základní představení chovatelské stanice, historie, filozofie chovu.
    
2. **Plemeno Shiba-Inu** – charakteristika plemene, povaha, nároky, vhodnost do rodiny.
    
3. **Nabídka štěňat** – dynamicky načítaný seznam štěňat s fotografiemi a popisy.
    
4. **Podmínky prodeje** – informace pro budoucí majitele, pravidla odběru štěňat.
    
5. **Kontakt** – adresa, telefon, e-mail + jednoduchý kontaktní formulář.
    
## **Návrh designu / wireframe**

- Barevné ladění v přírodních tónech (béžová, hnědá, jemná oranžová).
    
- Přehledné blokové uspořádání sekcí pod sebou.
    
- Karty štěňat v gridu (2–3 sloupce podle šířky obrazovky).
    
- Jednoduché a čitelné UI, moderní prvky (rounded, shadow, Tailwind komponenty).
    

_(Žák může doplnit ruční skicu, schéma nebo jednoduchý digitální wireframe.)_

## **Použité technologie**

- **Frontend:** HTML, CSS nebo Tailwind CSS, JavaScript
    
- **Data:** JSON soubory pro čtyři textové sekce + JSON nabídka štěňat
    
- **AJAX:** načítání obsahu sekcí a seznamu štěňat
    
- **Backend:** jednoduchý PHP endpoint pro odeslání kontaktního formuláře
    
- **Verzování:** Git + GitHub (repozitář dostupný vyučujícímu)
